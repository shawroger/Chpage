# Chpage
a light css library to hide div as a page


# install it

```

npm intall chpage --save

```

# load in script

```

<script type="text/javascript" src="index.js"></script>

```

# start a new example

```

import Chpage from 'chpage'

let my = Chpage.new()

/* use 'my' instead of 'Chpage' */

```
